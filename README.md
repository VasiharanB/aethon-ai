## 📖 Overview

Aethon AI is a comprehensive placement preparation platform designed to help students prepare for technical interviews through AI-assisted learning, coding assessments, aptitude tests, and company-specific practice.

The platform also provides recruiters and administrators with tools to create, manage, and evaluate assessments efficiently.
